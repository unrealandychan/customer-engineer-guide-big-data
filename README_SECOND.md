## Key Technical Terms to Know Cold

| Term | One-line definition |
|---|---|
| Exactly-once semantics | Pub/Sub = at-least-once; Dataflow deduplicates by record ID → exactly-once results in BigQuery |
| Watermark | Event-time point below which no late data is expected; window emits and closes after watermark passes |
| Canonical customer_key | SHA-256 hash of `(brand_id + source_customer_id)` — deterministic, portable identity |
| DataScan | Dataplex's scheduled data quality job — runs NOT_NULL, RANGE, FRESHNESS rules natively |
| ARIMA_PLUS | BigQuery ML time-series model — trains in SQL, no cluster, accessible to analysts |
| Flex Slots | BigQuery reserved capacity purchased for predictable/peak workloads (vs per-TB on-demand) |
