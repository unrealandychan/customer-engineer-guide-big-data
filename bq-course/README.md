# BigQuery Mini-Course

> **Source:** Built from the official Google Cloud BigQuery documentation at [cloud.google.com/bigquery/docs](https://cloud.google.com/bigquery/docs)  
> **Level:** Beginner → Intermediate  
> **Format:** 6 lessons — read in order. Each lesson has concepts, examples, and a practice exercise.

---

## Course Map

| Lesson | Title | Core skill |
|--------|-------|------------|
| [01](01-what-is-bigquery.md) | What is BigQuery? | Mental model, architecture |
| [02](02-storage.md) | How BigQuery Stores Data | Columnar format, table types, billing models |
| [03](03-tables-and-schema.md) | Tables, Schemas & Datasets | Create tables, data types, nested fields |
| [04](04-querying.md) | Querying with GoogleSQL | SELECT, aggregation, window functions, CTEs |
| [05](05-partitioning-and-clustering.md) | Partitioning & Clustering | Pruning, cost savings, choosing columns |
| [06](06-pricing-security-and-operations.md) | Pricing, Security & Operations | On-demand vs Editions, IAM, bq CLI |

---

## How to use this course

1. Read each lesson top to bottom
2. Run the SQL examples in the [BigQuery sandbox](https://cloud.google.com/bigquery/docs/sandbox) (free, no credit card)
3. Do the **Practice Exercise** at the end of each lesson before moving on
4. Come back to the Quick Reference tables when you forget syntax

---

## Pre-requisites

- Basic SQL knowledge (SELECT, WHERE, GROUP BY)
- A Google account (for the free sandbox)
